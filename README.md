# 🚀 Churn Predictor Pro: Customer Exit Analysis

This repository contains an end-to-end Machine Learning project to predict customer churn for a telecom company. It includes a data analysis pipeline and an interactive Streamlit dashboard for real-time predictions.

## 📊 Project Overview
The goal of this project is to identify customers who are likely to cancel their subscription. By analyzing factors like **tenure**, **contract type**, and **monthly charges**, the business can take proactive steps to retain high-risk customers.

## ✨ Features
- **Exploratory Data Analysis (EDA):** Detailed analysis of churn drivers using Seaborn and Matplotlib.
- **Interactive Dashboard:** A Streamlit-based UI with two main tabs:
    - **Business Insights:** View churn rates, average tenure, and distribution charts.
    - **AI Prediction:** Input customer details to get an instant risk assessment (High Risk vs. Low Risk).
- **Data Preprocessing:** Handled missing values in `TotalCharges` and performed One-Hot Encoding for categorical variables.

## 📂 File Structure
- `app.py`: The Streamlit web application code.
- `subscriber_exit_study (1).py`: Python script containing data cleaning, EDA, and preprocessing logic.
- `WA_Fn-UseC_-Telco-Customer-Churn.csv`: The raw dataset used for the study.
